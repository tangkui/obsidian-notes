# 5. 初始化店铺库存

- 磐石后台--->店铺中心--->线上店铺管理--->分配预留库存--->批量导入。

- 将已经初始化仓库库存的X码数据，在店铺进行库存分配。

- 导入模板示例如下：

  | 行号  | X码           | 分配库存 | 仓组名称        |
  | ----- | ------------- | -------- | --------------- |
  | 60584 | X001CP0000001 | 10000    | 磐石-电商销售仓 |
  | 60584 | X001CP0000002 | 10000    | 磐石-电商销售仓 |
  | 60584 | X001CP0000003 | 10000    | 磐石-电商销售仓 |
  | ...   | ...           | ...      | ...             |
